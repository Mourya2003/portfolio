3d Roating image gallery

This project is a simple 3D rotating image gallery created using HTML, CSS, and JavaScript. It allows users to navigate through images using previous and next buttons. The rotation effect is achieved by adjusting the 'rotateY' transformation property of the image container element. The rotation angle is incremented or decremented by 45 degrees each time a button is clicked, and there's also an automatic rotation every 3 seconds.
